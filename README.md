# image_to_ascii

Converts any image into ASCII text art. Dark pixels become dense characters, bright pixels become sparse ones, until the whole image is rebuilt out of text.

Comes in two forms:
- `main.py`, a Python command-line script
- `index.html`, a browser UI with drag-and-drop upload, no install needed
